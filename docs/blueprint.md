# **App Name**: Mapel: Map Towards Mastery

## Core Features:

- Role Selection & Login: Users select a role (Student, Guru, Creator) and log in with local storage credentials or continue as guest.
- Personalized Dashboard: Role-specific dashboard with progress tracking, learning playlists, and study paths.
- Content Display: Serve static JSON content. Display YouTube videos, notes, quizzes.
- Gamified Progress Tracking: XP, streaks, and level tracking with Framer Motion animations and reward chests. Uses local storage.
- Collaborative Study Rooms: Create or join study rooms with unique codes, shared dashboards, and local chat. Uses local storage.
- Career Roadmap Hub: Static JSON driven career roadmap exploration hub.
- AI-Simulated Motivation: Provide Weekly Insights that provide personalized AI motivation for each student.

## Style Guidelines:

- Primary color: Electric purple (#9D4EDD) to capture Gen-Z's bold, digital-native aesthetic.
- Background color: Dark, desaturated purple (#1A1423) for default dark mode with high readability.
- Accent color: Neon pink (#F72585) for highlighting interactive elements and creating visual excitement.
- Body and headline font: 'Space Grotesk' (sans-serif) for a computerized, techy feel.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use bold, chaotic icons that capture the Gen-Z futuristic vibe. Focus on abstract forms, electric glows, and glitch effects to make the icons more futuristic and visually striking.
- Emphasize a fast, social, and energizing feel through seamless transitions. Mimic familiar UX of popular social media or video-sharing apps (e.g. TikTok), in order to feel familiar and reduce the cognitive burden for Gen-Z learners.
- Implement smooth Framer Motion animations on every interaction to make the app dynamic.
